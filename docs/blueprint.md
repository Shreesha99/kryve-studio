# **App Name**: Kryve Studio

## Core Features:

- Homepage Showcase: Display KRYVE's best work with large images and project descriptions, and GSAP animations.
- About Section: Describe KRYVE's values and studio's expertise using elegant typography and layout.
- Services Offered: List the services offered by KRYVE, each described succinctly, and displayed in a minimal format.
- Contact Form: Enable potential clients to contact KRYVE through a simple form. No database. Sends emails directly to KRYVE.
- Dark/Light Mode Toggle: Allow users to switch between a dark and light theme, enhancing user experience based on preference.
- Blog Section: Generate articles with news about the company

## Style Guidelines:

- Primary color: Dark navy blue (#1A237E) to convey professionalism and sophistication.
- Background color: Very light gray (#F5F5F5), almost white, creating a clean, modern canvas. 15% saturation.
- Accent color: Teal (#008080), for highlighting key elements and calls to action, while maintaining a professional look.
- Headline font: 'Playfair' serif, with a high-end feel.
- Body font: 'PT Sans' sans-serif.
- Generous spacing throughout the site, to enhance readability and create a sense of openness and sophistication, inspired by towa-digital.com.
- Subtle GSAP animations on page load and scroll, for a premium, engaging user experience.
